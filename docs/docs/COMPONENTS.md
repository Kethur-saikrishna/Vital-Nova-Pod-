## Controllers
- ESP32 x2
- ESP32-CAM

## Health Sensors
- MAX30102
- DS18B20
- MPU6050
- GSR Sensor

## Environmental Sensors
- BME280
- MQ135
- Soil Moisture Sensor

## Audio System
- LM393 Sound Sensor
- DFPlayer Mini
- Speaker
- MicroSD Card

## Alerts
- Buzzer
- Traffic Light LEDs

## Power
- Lithium Battery
- TP4056 Charging Module

## Cooling System
- Relay Module
- Exhaust Fan
- Cooling Fan
